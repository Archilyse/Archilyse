Controls and Keyboard Shortcuts
Workflow
  • Select a feature type and click on the areas to be transformed into that type.
  • Press "Save" to save the changes and finish.
Select type
  • Click on a type name to select the type.
  • Press the underlined key from the name to select the type.
  • Press 1 to 9 to select the type indexes 1 to 9.
  • Press shift to select the type indexes 10.
  • Press shift + 1 to 9 to select the types indexes 11 to 19.
View controls
  • Mousewheel to zoom.
  • Right click and drag to pan.
  • Press 0 to reset zoom and panning (To center the screen).
  • Hold key x to see the original floorplan with full screen.
  • Press key y to open the original floorplan in a new window.
  • Hold space to hide the features and see the floorplan.
  • Hold m to hide the already classified areas.
