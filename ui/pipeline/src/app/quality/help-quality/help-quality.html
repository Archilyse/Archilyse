Instructions

Request the QA
  • First step is to have the basic features calculated. Press "Calculate Basic Features" to do so. It can take from 2 to 20-30 minutes depending on the size of the site. You can close this window and keep on working, if you leave the window open reload the window to check if the process finished.
  • Remember to tun "Recalculate Basic Feat." if there were changes in the pipeline
  • After the Basic features have been calculated, press "Run Qa Analysis" this process should take a few seconds only.
Make a review of the QA
  • Pressing the button "Save notes" will update the notes for the site in case of any problem detected.
  • Pressing the button "Validate" means the manual work over the site is complete and the simulations will be triggered.
